# 우리 반 랜덤 학생 뽑기

교실 TV 화면에 맞춰 큰 글씨로 학생을 무작위로 선정하는 단일 HTML 웹앱입니다. 엑셀 명단을 그대로 붙여넣어 등록하고, 한 번 뽑힌 학생은 다시 뽑히지 않도록 관리합니다.

## 주요 기능

- 엑셀에서 복사한 명단을 그대로 붙여넣어 등록 (줄바꿈·탭·쉼표·세미콜론 자동 분리, 중복 이름 제거)
- '두구두구' 연출과 함께 진행되는 무작위 추첨, 선정 시 축하 효과와 효과음
- 한 번 뽑힌 학생은 후보에서 제외하여 전원이 골고루 선정되도록 관리
- 전체 학생·남은 학생·뽑힌 학생 수를 실시간 표시
- 남은 학생 목록과 뽑힌 학생 순서 목록을 함께 제공
- 추첨 초기화(다시 뽑기), 전체 지우기, 소리 켜기·끄기 지원
- 교실 TV 화면을 고려한 큰 글씨 디자인과 반응형 레이아웃

## 실행 방법

별도의 설치나 빌드 과정이 없습니다. `index.html` 파일을 웹 브라우저에서 열면 바로 사용할 수 있습니다.

1. 명단 입력란에 학생 이름을 붙여넣습니다.
2. `명단 등록`을 누릅니다.
3. `두구두구 뽑기!`를 눌러 학생을 선정합니다.

## 기술 스택

- HTML / CSS / 순수 JavaScript (단일 파일, 외부 의존성 없음)
- Web Audio API (효과음 생성)
- Pretendard 웹폰트
